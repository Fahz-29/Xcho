# หน้าแก้ไขแบบฟอร์ม
หน้าแก้ไขแบบฟอร์ม ใช้สำหรับปรับปรุงหรือเปลี่ยนแปลงรายละเอียดของแบบฟอร์มที่มีอยู่เดิม ผู้ใช้สามารถเข้ามาจัดการโครงสร้างคำถาม แก้ไขข้อความ หรือดูข้อมูลเกี่ยวกับแบบฟอร์มได้ 

---
ถ้าผู้ใช้ต้องการจะแก้ไขแบบฟอร์ม สามารถกดแก้ไขได้ที่ปุ่ม **"แก้ไขฟอร์ม"** ของแบบฟอร์มที่เราเลือกต้องการจะแก้ไขได้ในแถบหน้าฟอร์มของฉัน

## ส่วนที่ 2 ดูข้อมูลเพิ่มเติมเกี่ยวกับแบบฟอร์ม
ในส่วนของหน้าเพิ่มเติมที่อยู่ทางขวาสุดของเว็บ ของปรับปรุงแบบฟอร์ม ที่จะแสดงขึ้นมาเฉพาะตอนที่ผู้ใช้กดปุ่ม**แก้ไขแบบฟอร์ม**  โดยผู้ใช้สามารถดูข้อมูลเพิ่มเติมของแบบฟอร์มได้ในหน้านี้ 

<div style={{ textAlign: 'center', marginBottom: '10px' }}>
  <img src="/img/editform/editform03.png" alt="คำอธิบายรูป" width="700" />
</div>

<div style={{ textAlign: 'center', marginBottom: '10px' }}>
  <img src="/img/create-form/create-form58.png" alt="คำอธิบายรูป" width="700" />
</div>

**รายละเอียดส่วนเพิ่มเติมแบบฟอร์ม** 
- **ดูผลลัพธ์** สามารถดูผลการตอบแบบฟอร์มทั้งหมดได้ในนี้ เช่น จำนวนคำตอบทั้งหมดที่ผู้ใช้ตอบแบบฟอร์มของเรา
- **แดชบอร์ด** ส่วนนี้จะเป็นการแสดงสถิติและการวิเคราะห์ข้อมูล โดยจะแสดงผลเป็นในรูปแบบกราฟและสถิติในนี้
- **ลิงก์/QR Code** ส่วนนี้กรณีที่ผู้ใช้เผยแพร่แบบฟอร์มเรียบร้อยแล้วผู้ใช้สามารถแชร์แบบฟอร์มด้วยลิ้งก์หรือ QR Code ได้
- **ลบแบบฟอร์ม** เมื่อผู้ใช้ต้องการที่จะลบแบบฟอร์มสามารถลบได้ในส่วนนี้ โดยแบบฟอร์มที่ถูกลบจะลบอย่างถาวร ไม่สามารถกู้คืนได้อีก
- **ข้อมูลแบบฟอร์ม** ตรงส่วนนี้จะแสดงข้อมูลอย่าง สถานะของแบบฟอร์ม , จำนวนคำตอบ , จำนวนผู้จัดการ

### 2.1 หน้าดูผลลัพธ์

**ภาพรวมของหน้าดูผลลัพธ์**
<div style={{ textAlign: 'center', marginBottom: '10px' }}>
  <img src="/img/editform/editform04.png" alt="คำอธิบายรูป" width="700" />
</div>

**ส่วนการกรองแสดงผลลัพธ์**
<div style={{ textAlign: 'center', marginBottom: '10px' }}>
  <img src="/img/editform/editform05.png" alt="คำอธิบายรูป" width="700" />
</div>
 
 - **ปุ่มดาวน์โหลด Excel , Csv :** ผู้ใช้สามารถเลือกดาวน์โหลดคำตอบของแบบฟอร์มในรูปแบบไฟล์ Exel หรือ Csv ได้ โดยกดที่ปุ่มคำว่า **"Exel"** หรือ **"Csv"** ได้ตามสะดวก
 - **ช่องค้นหา :** ส่วนนี้ไว้สำหรับการพิมพ์ค้นหากรณีที่ผู้ใช้ต้องการกรองดูคำตอบแบบเจาะจงก็สามารถพิมพ์ค้นหาได้ใน**ช่องพิมพ์ค้นหา**นี้ได้ 
 - **การกรองแสดงคำตอบ :** ถัดไปฝั่งทางขวาที่มีเป็นช่อง dropdown ให้ผู้ใช้เลือก เป็นการให้ผู้ใช้สามารถเลือกได้ว่าต้องการให้แสดงรายการคำตอบแบบฟอร์มกี่รายการในแต่ละหน้า
 - **การเลื่อนดูหน้าคำตอบแบบฟอร์ม :** ส่วนนี้ผู้ใช้สามารถกดเลื่อนดูรายการคำตอบแบบฟอร์มของแต่ละหน้าได้ โดยจะมีตัวเลขบอกว่าตอนนี้ผู้ใช้อยู่ในหน้าที่เท่าไหร่
   - **ปุ่ม**  <ChevronRight size={20} style={{ verticalAlign: 'middle', marginRight: '8px' }} /> **:**  ปุ่มกดไปหน้าถัดไปของรายการแสดงคำตอบแบบฟอร์ม
   - **ปุ่ม**  <ChevronLeft size={20} style={{ verticalAlign: 'middle', marginRight: '8px' }} /> **:**  ปุ่มกดไปหน้าก่อนหน้าของรายการแสดงคำตอบแบบฟอร์ม
   - **ปุ่ม**  <ChevronLast size={20} style={{ verticalAlign: 'middle', marginRight: '8px' }} /> **:**  ปุ่มกดไปหน้าสุดท้ายของรายการแสดงคำตอบแบบฟอร์ม
   - **ปุ่ม**  <ChevronFirst size={20} style={{ verticalAlign: 'middle', marginRight: '8px' }} /> **:**  ปุ่มกดไปหน้าแรกสุดของรายการแสดงคำตอบแบบฟอร์ม

**ส่วนแสดงรายการผลลัพธ์คำตอบแบบฟอร์ม**
<div style={{ textAlign: 'center', marginBottom: '10px' }}>
  <img src="/img/editform/editform06.png" alt="คำอธิบายรูป" width="700" />
</div>

- **แถบตัวกรองและหัวตาราง :** ในแต่ละคอลัมน์ (เช่น รหัสพนักงาน, ชื่อ-นามสกุล, ตำแหน่ง) ที่แสดงบอกในแต่ละช่องว่าข้อมูลรายการที่แสดงในแต่ละแถวที่อยู่ในหัวตารางนี้คืออะไร โดยจะมีเครื่องมือช่วยจัดการข้อมูล ดังนี้
  - **ช่องค้นหา :** ในส่วนช่องพิมพ์ค้นหานี้ผู้ใช้สามารถพิมพ์ข้อความเพื่อค้นหาข้อมูลเฉพาะเจาะจงที่ต้องการในคอลัมน์นั้นๆ ได้ทันที 
  - **เครื่องมือจัดการคอลัมน์ :**  
    - **ปุ่มเคลื่อนย้าย** <Expand size={20} style={{ verticalAlign: 'middle', marginRight: '8px' }} /> **:** ใช้สำหรับลากเพื่อสลับตำแหน่งคอลัมน์
    - **ปุ่มไปยังหน้ารายละเอียด (Move to Detail)** <MoveDownLeft size={20} style={{ verticalAlign: 'middle', marginRight: '8px' }} /> **:** ใช้สำหรับย้ายคอลัมน์นี้ไปแสดงผลในส่วนของ **"รายละเอียดเชิงลึก"** แทนการแสดงบนหน้าตารางหลัก 
    เมื่อผู้ใช้กดปุ่มไปยังหน้ารายละเอียดแล้ว สามารถกดปุ่ม **...** อยู่บริเวณซ้ายสุดของแต่ละแถว เพื่อดูรายละเอียดเชิงลึกได้ดีงตัวอย่างในภาพด้านล่างนี้
    <div style={{ textAlign: 'center', marginBottom: '10px' }}>
        <img src="/img/editform/editform07.png" alt="คำอธิบายรูป" width="600" />
    </div>

    - **ปุ่มซ่อนคอลัมน์ (Hide column)** <EyeOff size={20} style={{ verticalAlign: 'middle', marginRight: '8px' }} /> **:** ใช้สำหรับซ่อนคอลัมน์ที่เราไม่ต้องการจะให้แสดง

    - **ปุ่มจัดเรียงข้อมูล (Descending)** <ArrowDownNarrowWide size={20} style={{ verticalAlign: 'middle', marginRight: '8px' }} /> **:** ใช้สำหรับ **"จัดเรียงข้อมูล"** ภายในคอลัมน์ เช่น เรียงตามตัวอักษร ก-ฮ, เรียงตามรหัสพนักงานจากน้อยไปมาก หรือมากไปน้อย เพื่อให้ค้นหาข้อมูลที่ต้องการได้ง่ายขึ้น

- **ส่วนแสดงผลข้อมูล :** จะเป็นส่วนที่แสดงรายละเอียดรายการตามที่ผู้ใช้ตอบกลับแบบฟอร์มมาทั้งหมด

- **แถบเลื่อนด้านล่าง :** แถบสีม่วงด้านล่างสุดเป็นตัวแถบเลื่อนไว้สำหรับเลื่อนดูข้อมูลทางด้านขวาหรือซ้ายในส่วนของคอลัมน์ที่ผู้ใช้ต้องการจะเลื่อนดู

### 2.2 หน้าแดชบอร์ด
**ภาพรวมของหน้าแดชบอร์ด**

<div style={{ textAlign: 'center', marginBottom: '10px' }}>
  <img src="/img/editform/editform08.png" alt="คำอธิบายรูป" width="700" />
</div>

<br />

หน้านี้เป็นส่วนที่สรุปผลข้อมูลจากการตอบแบบฟอร์มออกมาให้เห็นเป็นรูปธรรม ช่วยให้ผู้ใช้วิเคราะห์ข้อมูลในภาพรวมได้ทันทีโดยไม่ต้องไล่ดูรายละเอียดทีละแถว
- **การวิเคราะห์ข้อมูล :** ผู้ใช้สามารถเลือกรูปแบบการแสดงผลสถิติที่ต้องการได้ผ่านแถบ Dropdown เพื่อดูข้อมูลในมุมมองที่แตกต่างกันได้
- **การล้างค่าตัวกรอง :** หากต้องการล้างการตั้งค่าการกรองทั้งหมด เพื่อกลับมาแสดงผลสถิติจากข้อมูลทุกชุดตามค่าเริ่มต้น สามารถกดปุ่ม **Reset Filter** ได้ทันที

### 2.3 หน้าลิงก์/QR Code

<div style={{ textAlign: 'center', marginBottom: '10px' }}>
  <img src="/img/editform/editform09.png" alt="คำอธิบายรูป" width="650" />
</div>

<br />

ส่วนการ **แชร์แบบฟอร์ม** ผู้ใช้สามารถส่งต่อแบบฟอร์มไปยังผู้ตอบได้สะดวกผ่าน 2 ช่องทางดังนี้
- **คัดลอกลิงก์ :** กดปุ่มคัดลอกเพื่อนำลิงก์ไปส่งต่อในช่องทางต่างๆให้ผู้ตอบแบบฟอร์ม
- **QR Code :** ดาวน์โหลดไฟล์รูปภาพ QR Code เพื่อให้ผู้ตอบแบบฟอร์มสแกนได้ทันที

### 2.4 หน้าลบแบบฟอร์ม

<div style={{ textAlign: 'center', marginBottom: '10px' }}>
  <img src="/img/editform/editform10.png" alt="คำอธิบายรูป" width="650" />
</div>

<br />

หากต้องการลบแบบฟอร์ม สามารถดำเนินการได้ดังนี้
- **การยืนยันลบ :** คลิกที่ปุ่ม **ลบแบบฟอร์มถาวร**  <Trash2 size={18} style={{ verticalAlign: 'middle' }} /> บริเวณมุมขวาล่างของหน้าต่างแจ้งเตือน
  - ⚠️ **ข้อควรระวัง :** เมื่อลบแล้ว ข้อมูลทั้งหมดจะถูกลบออกจากระบบอย่างถาวรและ ไม่สามารถกู้คืนได้

- **การยกเลิก :** หากต้องการยกเลิกการลบ สามารถเลือกทำได้ 2 วิธี
1. คลิกปุ่ม **ยกเลิก** ที่อยู่ด้านซ้ายของปุ่มลบแบบฟอร์ม
2. คลิกปุ่ม **ปิด** <X size={18} style={{ verticalAlign: 'middle' }} />  ที่มุมขวาบนของหน้าต่าง เพื่อกลับสู่หน้าจอปกติโดยไม่มีการลบข้อมูล

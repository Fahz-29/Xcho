# หน้ากล่องขาเข้า

หน้ากล่องขาเข้า ใช้สำหรับแสดงและจัดการรายการงานหรือแบบฟอร์มทั้งหมดที่ผู้ใช้เกี่ยวข้อง  
โดยระบบจะแบ่งการแสดงผลออกเป็นหลายส่วน เพื่อให้ผู้ใช้สามารถตรวจสอบ ค้นหา และจัดการงานได้อย่างสะดวก
<div style={{ textAlign: 'center' }}>
  <img src="/img/inbox/pageinbox.png" alt="ภาพ Dashboard หน้าหลัก" width="700" />
</div>

---

### ส่วนที่ 1 ภาพรวมสถานะงาน

ในส่วนบนของหน้ากล่องขาเข้า  
ระบบจะแสดงสถานะงานภาพรวมพร้อมจำนวนงานของแต่ละสถานะ  
ซึ่งจำนวนงานของแต่ละสถานะจะตรงกันกับที่แสดงในหน้าแรกของระบบ

<div style={{ textAlign: 'center' }}>
  <img src="/img/inbox/status.png" alt="ภาพ Dashboard หน้าหลัก" width="700" />
</div>


<div style={{ color: '#6A1B9A', fontWeight: 'bold', marginTop: '1rem' }}>
สถานะงานที่แสดงในภาพรวม
</div>

- **งานรอดำเนินการ**  
  คืองานที่ยังทำไม่เสร็จ รอการดำเนินการต่อ หรือรอการตอบแบบฟอร์มให้เสร็จสิ้น  
  พร้อมแสดงจำนวนงานในสถานะนี้

- **งานเสร็จแล้ว**  
  คืองานที่ดำเนินการเสร็จสิ้นหรือทำส่งเรียบร้อยแล้ว  
  พร้อมแสดงจำนวนงานในสถานะนี้

- **งานหมดเวลา**  
  คืองานที่เลยกำหนดระยะเวลาการทำหรือการส่งงานนั้นไปแล้ว  
  พร้อมแสดงจำนวนงานในสถานะนี้

---

### ส่วนที่ 3 การค้นหาและการกรองแบบฟอร์ม

ระบบได้จัดเตรียมเครื่องมือสำหรับค้นหาและกรองรายการแบบฟอร์ม  
เพื่อช่วยให้ผู้ใช้สามารถค้นหาแบบฟอร์มที่ต้องการได้อย่างรวดเร็ว

<div style={{ color: '#6A1B9A', fontWeight: 'bold' }}>
การค้นหาแบบฟอร์มตามตัวกรอง
</div>

<div style={{ textAlign: 'center' }}>
  <img src="/img/inbox/total.png" alt="ภาพ Dashboard หน้าหลัก" width="700" />
</div>

- **Dropdown เลือกสถานะแบบฟอร์ม**  
  ผู้ใช้สามารถเลือกแสดงแบบฟอร์มได้ตามสถานะ ได้แก่  
  ทั้งหมด, รอดำเนินการ, เสร็จแล้ว และสิ้นสุดแล้ว

- **การเลือกช่วงวันที่**  
  ผู้ใช้สามารถเลือกวันที่เริ่มต้น ถึง วันที่สิ้นสุด  
  เพื่อแสดงรายการแบบฟอร์มตามช่วงวันที่ที่ระบุ

<div style={{ color: '#6A1B9A', fontWeight: 'bold', marginTop: '1rem' }}>
การค้นหาแบบฟอร์มจากชื่อ
</div>

<div style={{ textAlign: 'center' }}>
  <img src="/img/inbox/search.png" alt="ภาพ Dashboard หน้าหลัก" width="700" />
</div>

- ผู้ใช้สามารถพิมพ์ชื่อแบบฟอร์มลงในช่องค้นหา  
  ระบบจะแสดงรายการแบบฟอร์มตามข้อความที่ค้นหา

### ส่วนที่ 4 การส่งออกข้อมูลแบบฟอร์ม

ในหน้ากล่องขาเข้า ผู้ใช้สามารถส่งออกรายการแบบฟอร์มออกมาเป็นไฟล์ได้  
โดยเลือกประเภทไฟล์จาก Dropdown ซึ่งประกอบด้วย
<div style={{ textAlign: 'center' }}>
  <img src="/img/inbox/export.png" alt="ภาพ Dashboard หน้าหลัก" width="700" />
</div>
- CSV  
- Excel  
- PDF  

ไฟล์ที่ส่งออกจะแสดงข้อมูลของแบบฟอร์ม ได้แก่  
ID, ชื่อหัวข้อแบบฟอร์ม, หน่วยงานผู้สร้าง, เบอร์โทรศัพท์, วันที่หมดเขต และสถานะของแบบฟอร์ม

### ส่วนที่ 6 การเลือกมุมมองการแสดงผลแบบฟอร์ม

ผู้ใช้สามารถเลือกมุมมองการแสดงผลรายการแบบฟอร์มได้ตามความถนัด ดังนี้

</div>

  เน้นการแสดงผลโดยใช้รูปภาพนำ  
  ข้อมูลจะถูกจัดเรียงเป็นบล็อก (Card)

<div style={{ textAlign: 'center' }}>
  <img src="/img/inbox/table.png" alt="ภาพ Dashboard หน้าหลัก" width="700" />
</div>
- **Table View**  
  เป็นมุมมองแบบตารางข้อมูล  
  เน้นการแสดงผลโดยใช้ข้อมูลตัวอักษรนำ  
  ข้อมูลจะถูกจัดเรียงเป็นแถว (Rows) และคอลัมน์ (Columns)

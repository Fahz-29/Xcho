# การเข้าสู่ระบบ

---

เมื่อผู้ใช้เข้าสู่เว็บไซต์ XCHO ในหน้าแรกของเว็บผู้ใช้จะเจอกับหน้า login / เข้าสู่ระบบ ให้ผู้ใช้กดที่ปุ่ม **“เข้าสู่ระบบด้วย PEA SSO”** ได้เลย

<img src="/img/login01.png" alt="คำอธิบายรูป" width="800" />

---

ในหน้าการเข้าสู่ระบบ ผู้ใช้สามารถลงชื่อเข้าใช้งานด้วยบัญชี PEA โดยการ กรอกข้อมูลที่ช่อง **“รหัสพนักงาน"** หรือ **"อีเมล”** และ ช่อง **“รหัสผ่าน”** จากนั้นเมื่อกรอกครบและตรวจสอบว่าถูกต้องแล้ว ให้กดปุ่ม **“เข้าสู่ระบบ”** เพื่อเข้าใช้งานเว็บไซต์ XCHO ได้เลย

<div style={{ textAlign: 'center' }}>
  <img src="/img/login02.png" alt="คำอธิบายรูป" width="350" />
</div>

<br />
<br />

กรณีที่ผู้ใช้ต้องการเข้าสู่ระบบด้วยช่องทางอื่น ผู้ใช้สามารถเข้าสู่ระบบโดยใช้ **ThaiID** ในการแสกนคิวอาร์โค้ดเพื่อเข้าสู่ระบบได้เช่นกัน

# การออกจากระบบ
การออกจากระบบ เป็นขั้นตอนสำคัญในการรักษาความปลอดภัยของบัญชีผู้ใช้งานและข้อมูลส่วนบุคคล เมื่อเสร็จสิ้นการปฏิบัติงานในระบบ XCHO ผู้ใช้ควรดำเนินการออกจากระบบทุกครั้ง เพื่อป้องกันการเข้าถึงข้อมูลโดยไม่ได้รับอนุญาต โดยเฉพาะอย่างยิ่งเมื่อมีการใช้งานผ่านอุปกรณ์สาธารณะหรืออุปกรณ์ที่ใช้งานร่วมกับผู้อื่น

---
ผู้ใช้สามารถไปที่หน้าการตั้งค่าได้โดยการกดไปที่รูปโปรไฟล์ที่มุมบนสุดด้านขวาของเว็บไซต์กดเลือกไปที่ **“ออกจากระบบ”** เพื่อออกกลับไปหน้าแรกของเว็บไซต์
